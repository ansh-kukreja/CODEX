# Python Notes

- Python is <b>Case Sensitive</b>

### Variable Names
- Can use Eng letters, Numbers, Underscores
- Identifier name can only start with a letter or underscore (Can't start with a Number)

<br>

### Data Types
1) Integer
2) String
3) Float
4) Boolean
5) None

