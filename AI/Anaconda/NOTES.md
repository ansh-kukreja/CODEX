# Anaconda

To create a virtual enviornment <br>
``conda activate``

To list all packages <br>
``conda list``

To create a new enviornment <br>
``conda create -n NAME``

To get list of Enviornments <br>
``conda env list``

To delete an enviornment <br>
``conda env remove -n NAME`` <br>
or <br>
``conda remove -n NAME --all``

To deactivate <br>
``conda deactivate``

To Open Jupyter Notebook <br>
``jupyter notebook``

To Start Jupyter Lab <br>
``jupyter lab``
