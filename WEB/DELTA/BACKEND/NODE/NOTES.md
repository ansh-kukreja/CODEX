## NODE.js NOTES

<b>NODE.js</b></br>It is a runtime enviornment of JavaScript that allows us to run JavaScript in our terminal like we do in our browser's console

<br>

### NPM Commands
- To Install Packages <br>
``npm install PACKAGE_NAME``

- To Initialize a Project <br>
``npm init``

- For Quick Initialization <br>
``npm init -y``
