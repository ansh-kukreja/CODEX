# Setup Notes

Step 1 - Download Faker Package
```
npm i @faker-js/faker
```

Step 2 - Download MySQL2 Package
```
npm i mysql2
```

Step 3 - Create a Database using MySQL Workbench.

Step 4 - Run this cmd in Project Terminal
```
/usr/local/mysql/bin/mysql -u root -p

Enter MySQL Password
```

<br/>
<br/>

- To Run a SQL Command file
```
1) Do Step 4

2) USE DATABASE name;

3) source file.sql
```
